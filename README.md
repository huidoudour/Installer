# huidoudour's Installer

## 特性介绍

### Shell 安装器
- Shizuku 授权的 Shell 安装器
- 定义了多个安装类型 (APK/APKS/XAPK)
- 定义了几个安装请求者参数，可通过设置调整
> 未进行过对 APKS&XAPK 的多种测试

### C++ 原生库
- ✅ 集成 C++ 共享库 (`libinstaller-native.so`)
- ✅ 支持架构: arm64-v8a, x86_64
> 实际并无有效功能，仅用于测试 C++ 库的集成。

### APP 本身

#### 语言切换
- ✅ 简体中文
- ✅ 英语翻译
